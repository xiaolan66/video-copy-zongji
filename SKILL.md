# 行业：国际物流 / 集运 / 大件运输

## ⚠️ 使用前必读：受众自动识别

生成文案前，先确认受众类型，再选择对应章节内容。

| 类型 | 受众 | 核心痛点 | 文案基调 |
|------|------|---------|---------|
| **A型** | 海外华人（英/加/美/澳等） | 怕被骗、怕临时加价、怕麻烦 | 直接、犀利、揭秘潜规则 |
| **B型** | 国内父母（为海外子女下单） | 怕操作复杂、怕弄丢、怕被税 | 温暖、保姆式、降低门槛 |

**快速识别法**：提及"帮孩子寄"、"给儿子女儿"、"国内爸妈" → B型；提及"从国内买"、"寄到英国/美国" → A型。

## 核心公式：五件套结构

每条爆款文案都遵循这个结构：

```
[开场钩子] → [痛点放大] → [解决方案] → [价格锚定] → [行动召唤CTA]
```

## 一、开场钩子（Hook）—— 0-3秒定生死

**目标**：让目标用户立刻感到"这说的是我"，产生停留。

### A型钩子（海外华人版）

#### 公式A：否定旧行为（最推荐）
```
"寄东西回国，千万别随便找一家快递！"
"在[国家]的华人，寄东西回国千万不要这样做了！"
"还在[错误做法]？只会花冤枉钱！"
```

#### 公式B：身份锚定
```
"在[国家]的华人注意了！"
"海外华人寄东西，这几个坑你踩过吗？"
"有[多少人]被快递坑过？"
```

#### 公式C：痛点提问
```
"寄东西回国，被临时加过价吗？"
"有没有被快递骗过？"
"[N]个华人问我同一个问题，今天统一回答："
```

#### 公式D：经验背书
```
"我做国际快递11年了，今天把里面的潜规则全说清楚"
"寄东西回国，这几个坑你必须知道"
```

#### 公式E：价格冲击
```
"同样寄15公斤，别人收你天价，我只收他三分之一"
"寄一次东西，运费比东西还贵，你遇到过吗？"
```

## 二、痛点放大（Pain Point）

### A型：海外华人痛点库（选2-3个，不要全说）

核心痛点词汇：
- **临时加价**（最常见骗局，海外华人最怕）
- **花冤枉钱**（损失厌恶，最强痛点）
- **货到仓库进退两难**（被动处境）
- **被骗找不到人**（安全感缺失）
- **不知道哪些能寄**（信息焦虑）
- **怕清关麻烦**（操作恐惧）
- **不知道寄一次多少钱**（价格焦虑）
- **英文不好怎么取件**（语言障碍）

句式模板：
```
"你以为[用户以为的正确方式]？真相是[揭露误区]"
"还在[用户当前做法]？只会[负面结果]"
"很多华人以为[误解]，其实[真相]"
```

海外华人专属痛点句式：
```
"货到仓库才告诉你，食品要加钱、药品要加钱——不退也得发。"
"被收了冤枉钱，货又不能不要，只能吃哑巴亏。"
"寄一次东西比东西本身还贵，贵得离谱。"
"找了好几家，问了半天，还是不知道到底要花多少钱。"
"清关那么复杂，英文不好怎么办？"
```

---

### B型：国内父母痛点库（选2-3个）

核心痛点词汇：
- **怕操作复杂**（畏难心理，手机上搞不定）
- **怕货丢了**（心疼东西，更心疼白折腾）
- **怕被税/扣关**（不知道能不能寄、会不会被海关扣）
- **怕孩子取不了件**（担心孩子英文不好没法取件）
- **怕流程太麻烦**（不知道怎么打包、找谁寄）
- **不知道该寄什么**（腊肉能不能寄？药品行不行？）

句式模板：
```
"想寄又不敢寄，怕丢、怕被税、怕孩子没法取。"
"叔叔阿姨您放心，咱们就是专门帮您解决这些麻烦的。"
```

## 三、解决方案（Solution）

### 服务内容库（选4-6个核心的，每次只讲最重要的）

| 服务 | 话术模板 |
|------|---------|
| 真实报价 | "全程真实报价，不玩低价引流，说多少就是多少" |
| 无隐藏费用 | "不搞隐形消费，没有临时加价" |
| 免费仓储 | "免费仓储，存多久都不收钱" |
| 免费打包材料 | "纸箱、气泡膜、胶带、真空袋全部免费提供" |
| 免费拍照验货 | "包裹到仓先拍照给你看，有问题直接帮退" |
| 免费拆箱抽真空 | "帮你拆箱整理、抽真空压缩，省空间又防潮" |
| 上门取件 | "国内上门取件，你在家等着就行" |
| 木架木箱加固 | "大件用木架木箱加固，保护得严严实实" |
| 双清包税 | "双清包税，零套路，零隐藏费用" |
| 派送到门 | "直接派送到你家门口 / 孩子家门口" |
| 中文客服 | "中文客服全程跟踪，随时联系" |
| 快速时效 | "快至5天签收，最慢25-30天" |

### 话术原则
- **具体不抽象**："15公斤箱子"比"大件"更可信
- **用实物代替品类**："腊肉、腊肠、药品、麻将机、冰箱"比"食品、电器"更真实
- **数字量化**："快至5天"比"很快"更可信
- **消除语言障碍**（A型）："全程中文客服，英文不好也完全没问题"
- **消除操作门槛**（B型）："您把东西寄到仓库，剩下的全交给我们"

## 四、价格锚定（Price Anchor）

### 锚定三段论（让价格显得超值的核心）

**A型：**
1. 揭露骗子价格：报低价→临时加价→实际比正规渠道还贵
2. 建立市场高价参照：普通转运"几公斤收几十公斤运费"
3. 呈现我们的价格：明确具体，不玩套路

**B型：**
1. 建立情感价值参照：一顿饭钱换孩子开心几个月
2. 对比当地购买价：在英国/加拿大买要"大几千英镑/加币"
3. 零隐藏消费的安心感："一分钱杂费没有"

### 核心价格话术

```
"运费只要一顿饭钱"
"15公斤箱子寄伦敦/曼彻斯特只要390元"
"低至8元/公斤"
"[N]公斤箱子全包（含打包、清关、派送）只要[N]元"
```

### 价格展示格式（可选）

**格式1：按城市一口价**
```
"15kg箱子寄伦敦曼彻斯特390元，伯明翰利兹380元，利物浦爱丁堡370元"
```

**格式2：单价+对比**
```
"海运每公斤只要8元，比你飞机托运便宜多了"
```

**格式3：总价+服务打包**
```
"15公斤箱子全包（含打包、清关、派送）只要400元左右"
```

 在评论区留下'孩子在哪个城市'，我把详细价格发给您。"

### B型次选CTA
```

```

### CTA设计原理

| 要素 | 作用 |
|------|------|
| 点红心+评论 | 提高视频互动权重，算法推荐更多曝光 |
| 评论地区/我要寄 | 精准销售线索（地区+意向+主动询问） |
| 私信发价格表 | 自动触达，低成本转化 |
| "三个字"或城市+重量 | 降低行动门槛，提升执行率 |
| 本人承诺（A型） | "我本人看到"增加信任感和承诺感 |
| 关注引导（B型） | 降低门槛，鼓励先关注再私信 |

## 十、高转化脚本检验清单

写完脚本后，逐项检查：

- [ ] **开场3秒**：是否有一个让目标用户立刻感到"说的是我"的钩子？
- [ ] **受众匹配**：是否针对正确的受众类型（A型海外华人 / B型国内父母）？
- [ ] **痛点真实**：用户是否真的经历过这个痛点？
- [ ] **方案具体**：是否有具体服务项目（不是笼统说"很方便"）？
- [ ] **价格可视**：是否用"一杯奶茶钱"/"一顿饭钱"等类比降低价格感知？
- [ ] **数字说话**：是否有具体重量、具体城市、具体价格？
- [ ] **CTA明确**：是否给了一个低门槛的、评论留资的行动指令？
- [ ] **场景真实**：是否用了具体的物品名（腊肉/冰箱/麻将机）？
- [ ] **无绝对承诺**：时效是否用"快至N天"而非"保证N天"？
- [ ] **口语化**：句子是否短（15字以内），适合口播？
