# 视频号爆款口播文案生成器
# 行业：国际物流 / 集运 / 大件运输
# 受众：海外华人华侨（英国市场为例）

## 核心公式：五件套结构

每条爆款文案都遵循这个结构，掌握后可快速生成高质量脚本：

```
[开场钩子] → [痛点放大] → [解决方案] → [价格锚定] → [行动召唤CTA]
```

---

## 一、开场钩子（Hook）—— 打破用户认知

**目标**：在0-3秒内让用户感到"这说的是我"，产生停留

### 公式A：否定旧行为（最常用）
```
"千万别再[用户正在做的错误行为]了！"
例："英国的华人千万别去当地华人超市购物了！"
例："从国内寄东西到英国，千万不要这样做了！"
```

### 公式B：揭露真相
```
"不会还有[目标人群]不知道，[他们应该知道的事]吧？"
例："不会还有英国华人不知道，网购商品可直接寄海外家中吧？"
```

### 公式C：紧急警告
```
"[目标人群]请注意！/ 千万要注意了！"
例："英国华人朋友一定要注意了！"
```

### 公式D：身份标签+冲突
```
"在[地点]的[身份描述]居然[意外结果]，就因为[他们做的事]"
例："在英国生活了20年的华人居然被邻居投诉了，就因为他们家全屋家具都是从中国网购的"
```

### 公式E：对比冲击开场
```
"在英国买[某物]要花[高价]，但其实……"
例："在英国买一台麻将机的价格，在国内能买3-5台"
```

### 公式F：提问式悬念
```
"[N]个[目标人群]问我同一个问题，今天统一答复："
例："这几天非常多英国华人问我：腊肉可以寄吗？药品可以寄吗？"
```

## 二、痛点放大（Pain Point）—— 让用户感到"被戳中"

在钩子之后，用2-3句话描述用户曾经的糟糕体验或当前困境：

**核心痛点词汇**（根据业务选2-3个）：
- "天价运费"（形容别人的贵）
- "几公斤东西被收几十公斤运费"（锚定行业乱象）
- "花冤枉钱"（损失厌恶）
- "自己扛行李"（体力痛苦）
- "在当地买太贵了"（价格对比）
- "不知道可以这样寄"（信息差）
- "等太久"（时效焦虑）
- "怕被税/怕扣关"（清关恐惧）

**句式模板**：
```
"你以为[用户以为的正确方式]？真相是[揭露误区]"
"还在[用户当前做法]？这样做只会[负面结果]"
```

## 三、解决方案（Solution）—— 集运服务介绍

### 服务内容库（根据实际选4-6个）

| 服务 | 话术模板 |
|------|---------|
| 免费仓储 | "免费仓储，存多久都不收钱" |
| 免费打包材料 | "纸箱、气泡膜、胶带、真空袋全部免费提供" |
| 免费拆箱抽真空 | "帮你拆箱整理、抽真空压缩，省空间又防潮" |
| 免费拍照验货 | "包裹到仓先拍照给你看，有问题直接帮退" |
| 上门取件 | "国内上门取件，你在家等着就行" |
| 木架木箱加固 | "大件用木架木箱加固，保护得严严实实" |
| 双清包税 | "双清包税，零套路，零隐藏费用" |
| 派送到门 | "英国全境直接派送到你家门口" |
| 快速时效 | "快至5天签收，最慢25-30天" |

### 话术原则
- **具体不抽象**："15公斤箱子"比"大件"更可信
- **用实物代替品类**："腊肉、腊肠、药品、麻将机、冰箱"比"食品、电器"更真实
- **数字量化**："快至5天"比"很快"更可信

## 四、价格锚定（Price Anchor）—— 让用户觉得超值

### 核心价格话术
```
"运费只要一杯奶茶钱"
"15公斤箱子寄伦敦/曼彻斯特只要390元"
"低至8元/公斤"
```

### 价格展示格式（可选）
**格式1：按城市一口价**
```
"15kg箱子寄伦敦曼彻斯特390元，伯明翰利兹380元，利物浦爱丁堡370元"
```

**格式2：单价+对比**
```
"海运每公斤只要8元，比你飞机托运便宜多了"
```

**格式3：总价+服务打包**
```
"15公斤箱子全包（含打包、清关、派送）只要400元左右"
```

### 锚定三段论（让价格显得超值的核心）
1. **建立高价参照**：在英国买要"大几千英镑"
2. **揭露错误价格**：普通转运"几公斤收几十公斤运费"
3. **呈现我们的价格**："只要一杯奶茶钱"

## 五、行动召唤CTA（Call to Action）—— 把互动变成销售线索

### 首选CTA模板（评论获客型）
```
"只要给这条视频点个红心，评论区留言[收货地区/我要寄/价格表]，
 我立刻把专属价格表发给你。"
```

### 次选CTA模板
```
"评论区留言'我要寄'三个字，我马上为您报价"
"在评论区留下'价格表'三个字，马上为您报价"
```

### CTA设计原理
| 要素 | 作用 |
|------|------|
| 点红心+评论 | 提高视频互动权重，算法推荐更多曝光 |
| 评论地区/我要寄 | 精准销售线索（地区+意向+主动询问） |
| 私信发价格表 | 自动触达，低成本转化 |
| "三个字" | 降低行动门槛，提升执行率 |

### 进阶CTA（适合已有粉丝基础）
```
"觉得有用就转发给身边需要的朋友"
"关注我，了解更多国际快递冷知识"
```

## 七、可寄物品清单（口播举例）

根据实际业务选填，不要一次全说：

**食品类**：腊肉、腊肠、干货、特产、零食
**药品类**：常用药、中药、保健品
**日用品类**：化妆品、护肤品、衣服、鞋子
**大件类**：家具、家电、麻将机、电动车
**建材类**：装修材料、瓷砖、门窗
**特殊类**：沙发、床垫、餐桌椅

**口播技巧**：不要念清单，每次只说3-5个代表性物品，其余用"等"省略

## 八、高转化脚本检验清单

写完脚本后，逐项检查：

- [ ] **开场3秒**：是否有一个让目标用户立刻感到"说的是我"的钩子？
- [ ] **痛点真实**：用户是否真的经历过这个痛点？
- [ ] **方案具体**：是否有具体服务项目（不是笼统说"很方便"）？
- [ ] **价格可视**：是否用"一杯奶茶钱"等类比降低价格感知？
- [ ] **数字说话**：是否有具体重量、具体城市、具体价格？
- [ ] **CTA明确**：是否给了一个低门槛的、评论留资的行动指令？
- [ ] **场景真实**：是否用了具体的物品名（腊肉/冰箱/麻将机）？
- [ ] **无绝对承诺**：时效是否用"快至N天"而非"保证N天"？
- [ ] **口语化**：句子是否短（15字以内）、适合口播？
- [ ] **情绪节奏**：是否有否定词（千万别/千万不要）制造紧迫感？
